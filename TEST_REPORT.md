## 一、项目概述

VoiceInk 是一款基于本地离线 ASR 模型（sherpa-onnx）的语音转文字桌面工具，支持按住快捷键说话、松开后自动将语音转为文字并粘贴到光标位置。

### 1.1 核心功能清单

| 功能模块 | 功能描述 | 状态 |
|---------|---------|------|
| 全局快捷键监听 | 按住 Ctrl+Space 录音，松开识别 | ✅ 已实现 |
| 麦克风录音 | 16kHz 单声道音频采集 | ✅ 已实现 |
| 语音识别 | 7 款 ASR 模型支持 | ✅ 已实现 |
| LLM 润色 | 可选大模型 API 润色文字 | ✅ 已实现 |
| 自动粘贴 | 跨平台文字粘贴到光标位置 | ✅ 已实现 |
| 声波动画 | 录音状态实时反馈 | ✅ 已实现 |
| 系统托盘 | 托盘图标、右键菜单 | ✅ 已实现 |
| 设置窗口 | 通用/模型/润色/关于 四页设置 | ✅ 已实现 |
| 模型下载 | HuggingFace 模型下载管理 | ✅ 已实现 |
| 提示音 | 录音开始/结束/错误提示音 | ✅ 已实现 |
| 开机自启 | Windows 注册表自启配置 | ✅ 已实现 |

## 二、代码架构测试分析

### 2.1 整体架构

```
App (中央协调器)
├── HotKeyManager (全局快捷键)
├── AudioRecorder (麦克风录音)
├── SpeechRecognizer (语音识别)
├── TextPolisher (LLM 润色)
├── TextPaster (文字粘贴)
├── SoundManager (提示音)
└── UI
    ├── FloatingWindow (悬浮窗)
    ├── TrayIcon (系统托盘)
    └── SettingsWindow (设置窗口)
```

**评估：** ✅ 架构清晰，模块职责明确，采用 PyQt6 信号/槽机制实现松耦合。

### 2.2 线程模型分析

| 线程/Worker | 模块 | 职责 | 线程安全 |
|------------|------|------|---------|
| 主线程 | App, UI | UI 操作和信号路由 | ✅ |
| sounddevice 回调 | AudioRecorder | 实时采集麦克风数据 | ✅ 使用 threading.Lock |
| pynput 监听 | HotKeyManager | 全局键盘事件监听 | ✅ 使用 threading.Lock |
| ModelLoadWorker | SpeechRecognizer | 模型加载 | ✅ Qt 信号通信 |
| TranscribeWorker | SpeechRecognizer | 语音转文字推理 | ✅ Qt 信号通信 |
| PolishWorker | TextPolisher | LLM API 调用 | ✅ Qt 信号通信 |
| ModelDownloadWorker | SpeechRecognizer | 模型文件下载 | ✅ Qt 信号通信 |

**评估：** ✅ 所有后台线程通过 Qt 信号（QueuedConnection）与主线程通信，线程安全设计良好。
