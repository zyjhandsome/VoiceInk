# Archived HTML visual reference

These five local HTML files are the approved visual reference for the native
PyQt6 settings and history surfaces. They are prototype artifacts only: the
VoiceInk runtime does not load them and has no WebEngine/WebChannel dependency.

The corresponding native surfaces live in:

- `voiceink/ui/settings_window.py`
- `voiceink/ui/settings_components.py`
- `voiceink/ui/history_window.py`
