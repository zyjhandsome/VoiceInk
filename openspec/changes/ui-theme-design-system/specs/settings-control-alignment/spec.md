## ADDED Requirements

### Requirement: 结构对齐在双主题下成立

主规格中「历史数值控件等宽同款」「右侧控件列右缘对齐」「设置页滚动条按需显示」的要求 SHALL 在有效主题为 `light` 与有效主题为 `dark` 时均成立。换肤不得破坏上述结构验收。

#### Scenario: 暗色下历史数值控件等宽

- **WHEN** 有效主题为 `dark`，且用户打开设置 → 通用页并滚动到「历史」
- **THEN** 「保留天数」与「最大会话数」步进框的外框宽度相等

#### Scenario: 暗色下开关与数值框右缘对齐

- **WHEN** 有效主题为 `dark`，且「保存语音历史」开关与「保留天数」步进框同时可见
- **THEN** 两者外框右缘与卡片右内边距对齐（允许 ±1px 渲染误差）

#### Scenario: 暗色下滚动条仍按需显示

- **WHEN** 有效主题为 `dark`，且某设置分页内容高度不超过可视区域
- **THEN** 垂直滚动条不占据内容右缘（按平台 AsNeeded 行为）
